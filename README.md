# Usage
## Pin connection
![SNOC-BRKWS01 pinout](BRKWS01.png)
Connect breakout board to the Arduino as this:

* Board -> Arduino
* 3,3V  ->  3,3V
* GND   ->  GND
* RX    ->  11
* TX    ->  10

## Example
Just run the example sketch.
